# Dutch Dictation Flow

这是一个适合部署到 GitHub Pages 的纯静态前端项目。

## 功能

- 创建文章，无需上传文章级音频
- 为每句话上传对应音频文件
- 为每句话添加荷兰语句子和中文翻译
- 支持按句子手动调整顺序（上移 / 下移）
- 按句子播放音频
- 听写答案并给出相似度评分
- 累计记录复习次数、句子熟练度、文章熟练度
- 标签、难度、来源信息支持

## 使用方式

1. 将仓库直接发布到 GitHub Pages
2. 打开 `index.html`
3. 新建文章
4. 添加句子文本、中文翻译、上传对应句子音频
5. 在“开始复习这篇文章”中听写、提交、进入下一句

## 数据存储

- 文章与句子文本、标签、复习记录、错题本复习记录、全文默写记录保存在浏览器 `localStorage`
- 句子音频文件保存在浏览器 `IndexedDB`
- 这两种存储方式都会在刷新或关闭页面后保留数据

## 备份与迁移

- 当前项目已增加“数据备份”页面，可导出整个应用数据为 JSON 文件
- 也可以通过该页面导入已保存的备份文件，恢复文章、句子、音频、错题本复习记录与全文默写记录

## 注意

- 若浏览器缓存被清除、浏览器配置重置或换设备，浏览器本地数据会丢失
- 请使用备份功能定期导出数据，确保不会丢失学习内容
